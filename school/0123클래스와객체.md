### @class 

해당 클래스의 존재를 알려준다. "그런게 있다" 순환참조에서 쓴다.    
@class 를 하면 반드시 .m 에서 #import를 해야한다.   
.m파일은 보여주지 않는다. 상속은 언제다 .h을 본다

### #import

## 선언부 - interface
- 클래스 
- 객체와 통신할 메서드
- 프로퍼티변수를 선언



## 구현부 - implementation
- 클래스의 메서드를 구현하는 방법(실제코드)가 담겨있다.
- .m파일에만 작성된다

함수는 어떠한 행위를 하는 것
구현부는 어떠한 행위를 할지 결정짓는 것
함수  NS~~~는 @interface에 들어가서도 안되며 반드시 { } 안에 들어가야 한다.
